框架搭建步骤:
1、下载app 包
https://github.com/webix-hub/jet-start/archive/master.zip
2、解压，把解压后的内容放到前段项目根目录下，如该项目的app目录
3、cd app
4、npm install 
5、npm start  启动  端口号默认8080
